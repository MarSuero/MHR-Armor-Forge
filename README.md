# MHR 外观mod替换工具

怪物猎人崛起（Monster Hunter Rise）外观mod骨骼/模型一键替换工具

## 功能

- **MOD修改**：支持 .zip / .7z / .rar 格式压缩包，自动替换骨骼和模型
- **存档备份**：备份和恢复 Steam 版怪物猎人崛起存档
- **自定义备份目录**：可设置存档备份的保存位置
- **拖放支持**：支持文件拖放导入
- **装备数据**：内置装备编号对照表

## 使用方法

1. 下载 `MHR_Mod_Tool_v1.4.exe`
2. 运行程序
3. 导入mod文件或进行存档备份

## 系统要求

- Windows 10/11
- 无需安装 Python
